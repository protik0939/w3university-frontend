### âœï¸ Creating a Blog

1. Click "New Blog" in sidebar
2. Fill required fields (marked with *):
   - Title (English) - Required
   - Content (English) - Required
3. Fill bilingual fields:
   - Title (Bengali)
   - Content (Bengali)
   - Excerpt (both languages)
   - Category (both languages)
   - Author (both languages)
4. Add tags:
   - Type tag and press Enter or click "Add"
   - Remove with X button
5. Set metadata:
   - Read time (e.g., "5 min read")
   - Image URL
   - Published date
   - Status (Draft/Published)
6. Click "Create Blog"

**Tips:**
- Slug auto-generates from English title
- Draft first, then publish when ready
- Use meaningful tags for better discovery
